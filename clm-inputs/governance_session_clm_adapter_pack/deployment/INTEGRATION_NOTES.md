# Integration Notes

## CIT Meeting → milestone verification support
Use an approved Governance Session resolution as supporting evidence before moving a Task
toward 'Verified by CIT'.

## Inspection Session → acceptance/rejection support
Use approved Session Resolution records to support milestone acceptance or rejection review.

## Dispute Session → stop-work recommendation support
Use an approved 'Recommend Stop Work' resolution to support the downstream Accounting Officer review.

## Monitoring Review → risk/action escalation support
Use approved resolutions and open Session Actions to feed monitoring escalations and remediation.
