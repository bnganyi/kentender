# KenTender Phase 1.5 Implementation Package
Public-sector Purchase Requisition / Demand Finalization module.

This package is a build-starting implementation set for Frappe/ERPNext custom app development.
It includes:
- starter DocType JSON definitions
- workflow JSON definitions
- Python scaffolding for validations, commitments, exceptions, and tender handoff
- permissions matrix
- report catalog
- UAT checklist

This package assumes the Phase 1 APP module already exists:
Procurement Plan -> Procurement Plan Item -> Purchase Requisition -> Tender
